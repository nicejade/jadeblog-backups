常见问题处理办法

git 时遇到fatal:multiple stage entries for merged file处理办法

  1. 执行 rm .git/index，回车
  2. 执行 git add -A，回车
  3. 执行 git commit -m "your commit message"

如何删除 Git 仓库所有提交历史记录

删除 .git 文件夹可能会导致您的 git 存储库中的问题。如果要删除所有提交历史记录，但将代码保持在当前状态，则执行以下操作非常安全：

1, Checkout

2, Add all the files

3, Commit the changes

4, Delete the branch

5, Rename the current branch to master

6, Finally, force update your repository

PS：这不会保留你的旧的提交历史; 详情参见 how to delete all commit history in github? [duplicate]。
